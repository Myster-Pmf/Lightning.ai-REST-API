API Documentation & Examples

🚀 Quick Start Examples

Get Machine Types & Pricing

curl -F "auth_file=@auth.json" /api/v2/machine-types

Start Studio with T4 GPU

curl -F "auth_file=@auth.json" -F "machine_type=T4" /api/v2/start

Check Studio Status

curl -F "auth_file=@auth.json" /api/v2/status

Switch Machine Type (Seamless)

curl -F "auth_file=@auth.json" -F "machine_type=L4" /api/v2/switch-machine

Execute Command

curl -F "auth_file=@auth.json" -F "command=python --version" /api/v2/execute

🌐 JavaScript Examples

Get Machine Types

const formData = new FormData(); formData.append('auth_file', authFile); fetch('/api/v2/machine-types', { method: 'GET', body: formData }) .then(response => response.json()) .then(data => console.log(data));

Start Studio

const formData = new FormData(); formData.append('auth_file', authFile); formData.append('machine_type', 'T4'); fetch('/api/v2/start', { method: 'POST', body: formData }) .then(response => response.json()) .then(data => console.log(data));
